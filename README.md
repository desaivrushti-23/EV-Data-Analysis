# EV-Data-Analysis
# Electric Vehicle Population Data Analysis

## Project Overview
This project analyzes the Electric Vehicle (EV) Population dataset using Python to discover trends, visualize data, and build a Linear Regression model for predicting electric vehicle range. The project demonstrates data preprocessing, exploratory data analysis (EDA), visualization, and basic machine learning techniques.

## Objectives
- Analyze EV registration data.
- Identify the top EV makes and models.
- Visualize EV adoption trends over time.
- Examine EV distribution across counties.
- Analyze CAFV eligibility.
- Build a Linear Regression model to predict Electric Range.

## Dataset
**Dataset:** Electric Vehicle Population Data

The dataset contains information such as:
- County
- City
- State
- Model Year
- Make
- Model
- Electric Vehicle Type
- Electric Range
- CAFV Eligibility
- Vehicle Location

## Machine Learning
A Linear Regression model is used to predict the **Electric Range** of an electric vehicle using available features such as:
- Model Year
- Make
- Model
- Electric Vehicle Type
- CAFV Eligibility

## Key Findings
- EV adoption has increased significantly over recent years.
- Tesla is among the most popular EV manufacturers.
- EV registrations are concentrated in certain counties.
- Electric Range varies by vehicle type and model.
- Linear Regression can be used to estimate Electric Range based on available vehicle features.
